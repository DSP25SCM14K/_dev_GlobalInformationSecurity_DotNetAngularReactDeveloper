# Dev Kumar - Global Information Security .NET Angular React Developer Portfolio

Dark, animated portfolio website tailored for a front-end-focused full-stack .NET / Angular / React role with emphasis on secure multi-tier applications, ASP.NET Core APIs, Entity Framework, SQL Server, component-based UI, CI/CD, and unit-tested Agile delivery.

## Built With

- HTML5
- CSS3
- JavaScript
- Canvas animation
- SVG architecture diagram
- Generated hero image
- Lucide icons

## Local Preview

Open `index.html` directly in a browser.

## Research Inputs

- ASP.NET Core and .NET application patterns for web APIs and multi-tier services.
- Entity Framework Core and SQL Server role signals: relational modeling, migrations, LINQ, indexing, transactions, query tuning, and data-access patterns.
- Angular and React role signals: component-based UI, TypeScript, responsive interfaces, reusable design systems, and modern front-end delivery.
- Security and SDLC signals: OWASP Top 10 awareness, authentication, authorization, validation, unit testing, CI/CD, Git, and Agile collaboration.

## Deployment

This repo is designed for GitHub Pages from the root of the default branch.
